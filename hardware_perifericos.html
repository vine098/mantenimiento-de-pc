Periféricos del equipo

Los periféricos como teclado, mouse y monitor también requieren mantenimiento para garantizar buen funcionamiento.

La suciedad, golpes o cables dañados pueden generar fallas simples pero frecuentes.

Este apartado cubre los cuidados esenciales.

Teclado

→ Sacudir boca abajo para liberar migas/polvo.

→ Aire comprimido entre teclas.

Mouse

→ Quitar suciedad del sensor con un hisopo.

→ Limpiar skatez (patitas) para mejorar deslizamiento.

Monitor

→ Pasar paño de microfibra seco, luego otro apenas humedecido.

→ Nunca usar limpiavidrios directamente sobre la pantalla.

Cables

→ Revisar que no haya conectores flojos ni cables rotos.

→ Evitar doblar cables en ángulos bruscos.

Este mantenimiento evita desconexiones inesperadas y fallas intermitentes.
